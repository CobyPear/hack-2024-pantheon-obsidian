    Steps

    Using a new or existing vault, manually add the plugin

    1. Visit the plugin release page
    2. Click the pantheon-obsidian-hack-20204-plugin.zip link to download the plugin files
    3. Unzip the files and move them into <your vault>/.obsidian/plugins/hack-2024-obsidian-plugin. Note the . in the obsidian dir. You may need to show hidden files if using a GUI
    4. In Obsidian, open the Settings
    5. Open Community Plugins and turn them on if they are not available. You may need to restart Obsidian to see the plugin show up in the list
    6. Once you see the plugin in the list, flip the switch on. You should see a new rocket button on the left ribbon. Installation is complete!
